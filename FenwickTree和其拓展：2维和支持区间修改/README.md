二维的FenwickTree沿用了一维的算法，只是添加了一维而已计算时候用二维的前缀和公式计算即可



支持区间修改需要维护的是差分数组吧b[i]=a[i]-a[i-1];而a[i]=sum(b[1]...b[i]);同样sum(a[1]..a[n])=n*sum(b[1]..b[i])-sum(b[i]*(i-1))
